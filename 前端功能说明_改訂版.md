# 前端功能说明（改訂版）

本文は `vue3-pyh` フロントエンドコードに基づき、GIS専門性を強化・再整理したシステム機能仕様書です。

---

## 1. システム概要

本システムは **Vue 3 + Vite + Element Plus + OpenLayers + ECharts** をベースとした鄱陽湖流域の生態環境モニタリングプラットフォームです。水文解析、土地被覆変化、温度・降水分布、RSEI生態評価、および空間回帰モデリングを統合的に提供します。

### 技術スタック

| 項目 | 技術 |
|------|------|
| フロントエンドフレームワーク | Vue 3 + Vite |
| UIコンポーネント | Element Plus |
| 地図エンジン | OpenLayers 7+ |
| チャートライブラリ | ECharts 5 |
| 空間データ形式 | GeoJSON / WMS / WFS / Shapefile |
| 状態管理 | Pinia |

---

## 2. ページ機能

### 2.1 ホームページ（首页）

- システムタイトル・概要の表示
- 左側ナビゲーションによる各業務ページへの統一導線
- 未ログイン時はログインページへ自動リダイレクト
- ヘッダーにログインユーザーのアバター（頭文字）を表示

---

### 2.2 ログイン／ユーザー管理

- ログイン・新規登録のタブ切り替えフォーム
- `localStorage` を用いたフロントエンド認証状態の管理（`token`・`username` 保存）
- `request.js` が `Authorization` ヘッダーを自動付与
- 401レスポンス時にローカルのログイン情報を自動削除してログインページへリダイレクト

> **備考：** 現フェーズはフロントエンドのみのモック認証。バックエンドログインAPIとの連携は今後の対応。

---

### 2.3 研究区域紹介（介绍页）

**既存機能（調整あり）**

- 鄱陽湖研究区域の概況テキスト表示
- 洪水期・干水期の写真スライドショー

**追加・強化機能**

- **OpenLayers ベースマップ上での研究区域可視化**
  - 鄱陽湖流域の行政区界（GeoJSON）をオーバーレイ表示
  - 湖岸線の季節変動を年別・季節別でアニメーション表示
  - 主要観測地点（気象ステーション、水文観測点）のポイントレイヤーを表示
- **標高データの3Dプレビュー（概略表示）**
  - DEM由来の陰影起伏をWMS経由で表示
- **地図上のインタラクション**
  - 観測地点クリックでポップアップ（地点名・座標・種別・観測期間）表示

---

### 2.4 水文統計（水文统计页）

**既存機能（維持）**

- 水深平均値・水面面積・水量・年間降水量のサマリーカード
- 年次・水文フェーズの選択フィルター
- ローカルCSVアセットによるチャート駆動
- 水量変化トレンド・水深等級分布チャート

**追加・強化機能**

- **時系列アニメーション**
  - 年次別の水面積変化を地図上でアニメーション再生
  - スライダーで任意年次に移動可能
- **空間統計サマリー**
  - 選択フェーズ・年次の水面積をポリゴン面積として算出・表示
  - 水深ラスターのヒストグラムをWMS `GetMap` + クライアント側集計で生成
- **水文フェーズ定義パネル**
  - 洪水期・干水期・平水期の定義（水位閾値）をインタラクティブに確認

---

### 2.5 土地被覆分析（土地覆盖页）

**既存機能（維持）**

- 3つの分析モード：時系列分析・比較分析・土地利用転換マトリクス
- 地図上の領域描画
- 指定年次の土地利用統計テーブル表示
- バックエンドAPI連携：`/api/timeseries`・`/api/compare`・`/api/transition`
- 比較指標の切り替え（面積・変化・異常）

**追加・強化機能**

- **地図連動強化**
  - 描画したポリゴンをGeoJSONとして保存・再読み込み可能
  - 描画領域内のWMSレイヤー（土地被覆分類ラスター）をリアルタイムにクリップ表示
  - 描画ツールに「円形」「バッファー作成」を追加
- **転換マトリクスの可視化改善**
  - サンキーダイアグラム（ECharts）による土地利用フロー表示
  - セルクリックで転換ポリゴンを地図上にハイライト
- **エクスポート機能**
  - 統計テーブルをCSV形式でダウンロード
  - 描画ポリゴンをGeoJSON形式でエクスポート

---

### 2.6 温度・降水分析（温度和降水页）

**既存機能（維持）**

- 温度／降水の切り替え
- 年次・月次データの閲覧
- 分析方式：領域描画分析・都市データ表示・単点分析・月次分析
- バックエンドAPI連携：`/api/RT/avg`・`/api/RT/code`・`/api/RT/coordinate`・`/api/RT/points`

**追加・強化機能**

- **空間補間マップ（クライアント側生成）**
  - 観測点データをIDW（逆距離加重）法でクライアント側補間し、疑似カラーマップをCanvasオーバーレイで表示
  - 補間解像度をスライダーで調整可能（粗→細）
- **異常値ハイライト**
  - 月次分析において平均±2σを超える月をチャート上で警告色表示
- **都市間比較チャート**
  - 複数都市を選択してレーダーチャートで年間気候プロファイルを比較
- **データダウンロード**
  - 選択期間・地点のCSVエクスポート

---

### 2.7 生態評価指数（生态评价指数页）

**既存機能（調整あり）**

- RSEI統計サマリーカード
- 地図クリックによる地点分析
- トレンド分析・比較分析・都市別比較
- 自由描画領域による比較
- バックエンドAPI連携：`/predict`・`/api/RegionCompare/geojson/`・`/api/RegionCompare/point/`
- ポップアップ：履歴トレンド・予測分析・異常検知

**追加・強化機能**

- **RSEI空間分布マップの強化**
  - WMS上のRSEIラスターに対してカスタムカラースケール（5等級分類）を凡例とともに表示
  - 年次スライダーで経年変化をアニメーション表示
- **予測結果の空間可視化**
  - `/predict` レスポンスをポイントレイヤーとして地図上にプロット
  - 信頼区間バンドをラインチャートで表示
- **ホットスポット検出パネル（新規）**
  - RSEI値が著しく低下した（生態劣化）グリッドセルを自動抽出して地図上にマーカー表示
  - 閾値スライダーで感度調整可能
- **レポートエクスポート（新規）**
  - 選択領域のRSEI統計・チャート画像を含むPDFサマリーをブラウザ印刷で出力

---

### 2.8 機制分析・空間回帰（机制分析页）

**既存機能（維持）**

- データ前処理
- 空間回帰分析の実行
- 結果CSV・Shapefileのダウンロード
- 結果の地図可視化
- 区域ファイルのアップロード
- バックエンドAPI連携：`/api/preprocess`・`/api/spatial-regression`・`/api/read-csv`・`/api/download-csv`・`/api/download-shapefile`・`/api/results/visualize`・`/api/shapefile-fields`・`/api/upload/region`

**追加・強化機能**

- **モデル設定UIの強化**
  - 空間回帰モデル種別の選択肢を明示：OLS・SLM（空間ラグ）・SEM（空間誤差）・GWR（地理的加重回帰）
  - 空間重み行列のタイプ選択（クイーン隣接・ルック隣接・距離バンド）
  - 帯域幅パラメーター（GWR用）のスライダー入力
- **診断統計パネル（新規）**
  - Moran's I（空間自己相関）・AIC・R²・残差分布ヒストグラムをダッシュボード形式で表示
- **GWR係数マップ（新規）**
  - GWR実行後、各説明変数の局所係数をコロプレスマップで地図上に表示
  - 変数切り替えセレクト付き
- **フィールド選択UIの改善**
  - Shapefileフィールド一覧（`/api/shapefile-fields`）をチェックボックスリストで表示し、目的変数・説明変数をドラッグ＆ドロップで割り当て可能

---

### 2.9 モニタリングデータ（监测数据页）

**既存機能（調整あり）**

- ドローン映像のリアルタイム表示
- 現地調査写真のスライドショー

**追加・強化機能**

- **観測地点マップ（新規）**
  - ドローン飛行軌跡（GeoJSON LineString）をOpenLayersで表示
  - 各撮影地点のサムネイル付きポップアップ
- **メタデータパネル（新規）**
  - 撮影日時・解像度・センサー種別・フライト高度をテーブル形式で表示
- **写真フィルター（改善）**
  - 日付範囲・ロケーション・調査種別でフィルタリング可能

## 4. データ・通信

- 統一リクエストユーティリティ：`src/utils/request.js`（Token自動付与・401ハンドリング）
- 地図ベースマップ・主題図レイヤー：GeoServer WMS
- **追加**：WFS `GetFeature` リクエストによる属性データ取得（フィルタリング・空間クエリ）
- **追加**：クライアント側GeoJSON処理にTurf.jsを採用（面積算出・バッファー・クリップ）
- 静的アセット（CSV・画像・動画）は引き続き一部ページで直接参照

---

## 5. GIS機能強化まとめ

本改訂版で新たに追加・強化した主なGIS専門機能：

- **地図インタラクション統一**：全ページ共通の描画ツールバー・レイヤーパネル・凡例コンポーネントを整備
- **空間補間の可視化**：観測点データのIDWクライアント側補間・疑似カラーオーバーレイ
- **GWR係数の空間表示**：局所係数のコロプレスマップによる地理的不均一性の視覚化
- **空間回帰診断統計**：Moran's I・AIC・R²のダッシュボード表示
- **ラスターアニメーション**：年次別RSEI・水面積の時系列アニメーション（WMSスライダー連動）
- **ホットスポット検出**：生態劣化グリッドの自動抽出・地図マーカー表示
- **GeoJSON入出力**：描画ポリゴン・分析結果のGeoJSON形式エクスポート／インポート
- **Turf.js統合**：面積・バッファー・クリップ等のクライアント側空間演算

## 6. 現フェーズの対象外事項

以下は現フェーズでは実装対象外（将来対応）：

- バックエンドとの完全認証連携（現状はフロントエンドモック）
- リアルタイムデータストリーミング（WebSocket）
- オフライン対応・PWA化
- 3D地形ビュー（Cesium連携）
