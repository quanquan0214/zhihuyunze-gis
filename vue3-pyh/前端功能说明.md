# 前端功能说明

本文基于当前 `vue3-pyh` 前端代码整理，描述系统已经实现的前端功能。

## 1. 系统概况

这是一个基于 Vue 3 + Vite + Element Plus + OpenLayers + ECharts 的生态环境监测前端，主要服务于鄱阳湖区域生态分析、土地覆盖分析、温度降水统计和空间回归建模。

## 2. 页面功能

### 2.1 首页

- 展示系统标题与简介
- 左侧导航统一进入各业务页面
- 未登录时自动跳转到登录页
- 顶部显示当前用户头像缩写

### 2.2 登录/注册

- 支持登录与注册两个表单页签
- 当前前端使用 `localStorage` 模拟登录态
- 登录后写入 `token` 和 `username`
- `request.js` 会自动携带 `Authorization` 请求头

说明：该页面当前是前端模拟流程，未直接调用后端登录接口。

### 2.3 介绍页

- 展示鄱阳湖研究区概况
- 展示洪水期、枯水期图片
- 使用底图展示研究区位置

### 2.4 水文统计页

- 展示水深平均值、水面面积、水储量、年降水量
- 支持选择年份和水文阶段
- 使用本地 CSV 资产驱动图表和统计展示
- 右侧图表包含：
  - 水储量变化趋势
  - 水深等级分布

### 2.5 土地覆盖页

- 支持三种分析模式：
  - 时序分析
  - 对比分析
  - 土地类型转换矩阵
- 支持在地图上绘制区域
- 支持查看指定年份的土地利用统计表
- 对接后端接口：
  - `/api/timeseries`
  - `/api/compare`
  - `/api/transition`
- 弹窗内支持切换对比指标：
  - 面积
  - 变化
  - 异常

### 2.6 温度和降水页

- 支持温度/降水切换
- 支持按年份、月份查看数据
- 支持三种分析方式：
  - 绘制区域分析
  - 显示城市数据
  - 单点分析
  - 月度分析
- 对接后端接口：
  - `/api/RT/avg`
  - `/api/RT/code`
  - `/api/RT/coodinate`
  - `/api/RT/points`
- 月度分析通过独立图表组件展示

### 2.7 生态评价指数页

- 显示 RSEI 统计卡片
- 支持地图点击点位分析
- 支持趋势分析、对比分析、城市对比分析
- 支持自定义区域绘制对比
- 对接后端接口：
  - `/predict`
  - `/api/RegionCompare/geojson/`
  - `/api/RegionCompare/point/`
- 弹窗展示：
  - 历史趋势
  - 预测分析
  - 异常检测

### 2.8 机制分析页

- 支持数据预处理
- 支持空间回归分析
- 支持结果 CSV 下载
- 支持 Shapefile 下载
- 支持结果地图可视化
- 支持上传区域文件
- 对接后端接口：
  - `/api/preprocess`
  - `/api/spatial-regression`
  - `/api/read-csv`
  - `/api/download-csv`
  - `/api/download-shapefile`
  - `/api/results/visualize`
  - `/api/shapefile-fields`
  - `/api/upload/region`

### 2.9 监测数据页

- 展示无人机实时监测视频
- 展示实地考察图片轮播

## 3. 共享组件

- `baseMap.vue`：OpenLayers 地图封装，支持 WMS 图层和 GeoJSON 叠加
- `RegionCompare.vue`：RSEI 区域对比图表弹窗
- `LandAnalysisDialog.vue`：土地覆盖分析结果弹窗
- `GlobalAlert.vue`：RSEI 点位历史趋势、预测与异常弹窗
- `AverageChart.vue`：温度/降水月度分析图表
- `timechart.vue`、`deepchart.vue`、`piechart.vue`、`barchart.vue`、`linechart.vue`、`rseiChart.vue`：各类 ECharts 图表组件

## 4. 数据与请求

- 统一请求封装位于 `src/utils/request.js`
- 自动注入 Token
- 401 时自动清除本地登录信息并跳转登录页
- 地图底图和专题图层主要依赖 GeoServer WMS
- 部分页面仍直接读取本地静态 CSV / 图片 / 视频资源

## 5. 当前前端功能总结

当前前端已经覆盖：

- 生态环境概览展示
- 水文统计分析
- 土地覆盖分析
- 温度与降水统计
- RSEI 指数分析与预测
- 区域机制分析与空间回归
- 监测数据展示
- 基础登录与权限控制
